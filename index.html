SciSpark

Distributed Computing for Large-Scale Climate Science and Analytics

SciSpark is a lightning fast big data technology.

It is a scalable system for interactive model evaluation and for the rapid developent of climate metrics and analysis to address the pain points in the current model evaluation process.
